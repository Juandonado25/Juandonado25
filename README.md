## Proyectos destacados

### Cybersecurity Homelab

Laboratorio de ciberseguridad que simula una infraestructura empresarial con segmentación de red (WAN, LAN, DMZ), firewall pfSense, Wazuh como SIEM y monitoreo de eventos en tiempo real.

Incluye:

* Recolección y análisis de logs
* Configuración de reglas de firewall
* Integración de herramientas de detección
* Simulación de escenarios de red

### SOC Practices

Repositorio de prácticas enfocadas en validación de reglas de seguridad y comportamiento de red dentro del laboratorio.

Incluye:

* Pruebas de segmentación de red
* Validación de tráfico permitido y bloqueado
* Verificación de eventos en SIEM

Colección de análisis y resolución de escenarios de seguridad basados en plataformas como Let's Defend y Hack The Box.

Incluye:

* Análisis de alertas
* Investigación de incidentes

* Aplicación de conceptos de MITRE ATT&CK
